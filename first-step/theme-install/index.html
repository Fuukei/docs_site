安装主题 ​

配置要求 2025年4月 ​

最低要求：

Nginx/Apache：2020 年之后发行的版本。
PHP：8.0 及以上版本
MySQL：5.6 及以上版本

推荐：

Nginx：1.23.X 版本
PHP：8.2.X 版本
MySQL：8.0.X 版本

上传并启用主题 ​

WARNING

下载主题后，请解压并将解压后的主题文件夹重命名为 “Sakurairo” ，然后重新压缩成 ZIP 格式文件（2.7.2及以下版本为必要步骤）。

进入仪表盘，点击左侧菜单的 外观 选项

点击主题页面的 安装主题

在刷新的页面点击 上传主题

然后的 上传主题 中，选中 桜いろ 主题安装

安装之后，启用主题
